Lists in HTML

Unordered Lists

These lists are used for items that do not have a particular order.
Types of unordered lists:
  • Disc
  • Circle
  • Square
  • None

Ordered Lists

These lists are used to emphasize the order of items.
Types of ordered lists:

    123

    ABC

    abc

    Roman Numerals

    Lowercase Roman Numbers

Nested Lists

We can place list inside another list to create a sublist.
  1. First Period
  2. Second Period
  3. Block Lunch
    1. A Block
    2. B Block
  4. Third period
  5. Fourth period
